fix(ui): Restaurer fonctionnalités complètes des widgets custom après migration EPIC-1

PROBLÈME
--------
La migration EPIC-1 avait simplifié à l'excès les widgets custom, causant des
régressions majeures dans QgsCheckableComboBoxLayer et
QgsCheckableComboBoxFeaturesListPickerWidget. Les fonctionnalités perdues
incluaient :
- Menu contextuel (Select All, géométrie)
- ItemDelegate custom (checkbox + icon + text)
- Event filters pour clic gauche/droit
- ListWidgetWrapper pour gestion features
- Filter debounce (300ms)
- Sort order (ASC/DESC)
- Font styling by state (checked/unchecked/filtered)

SOLUTION
--------
Restauration complète depuis before_migration/modules/widgets.py vers
ui/widgets/custom_widgets.py avec adaptations pour nouvelle architecture :

1. QgsCheckableComboBoxLayer (restauré à 100%) :
   - Retour à héritage QComboBox (au lieu de QgsCheckableComboBox)
   - Ajout ItemDelegate custom pour painting
   - Ajout menu contextuel avec sélection par géométrie
   - Ajout paintEvent pour affichage CSV
   - Ajout eventFilter pour gestion clics

2. QgsCheckableComboBoxFeaturesListPickerWidget (restauré à ~95%) :
   - Retour au layout QVBoxLayout (filter_le + items_le + list)
   - Ajout ListWidgetWrapper pour cache features
   - Ajout context menu (Select All/subset)
   - Ajout debounce timer (300ms) pour filtre texte
   - Restauration signatures complètes :
     * setLayer(layer, layer_props)
     * setDisplayExpression(expression) avec validation
     * setFilterExpression(filter_expression, layer_props)
     * setSortOrder(order='ASC', field=None)
     * currentSelectedFeatures() → list ou False
     * currentVisibleFeatures() → list ou False
   - Restauration signaux :
     * updatingCheckedItemList(list, bool) - signature complète
     * filteringCheckedItemList()

3. ListWidgetWrapper (nouvelle classe restaurée) :
   - Wrapper QListWidget avec métadonnées features
   - Stockage display_expression, filter_expression, subset_string
   - Listes features_list, visible_features_list, selected_features_list
   - Méthode sortFeaturesListByDisplayExpression(reverse=False)

4. ItemDelegate (nouvelle classe restaurée) :
   - Custom painting checkbox + icon + text
   - Support QStandardItem avec roles Qt
