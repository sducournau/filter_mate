fix(ui): Eliminate silent delegation failures in layout managers - v4.0.6

CRITICAL FIX: All .apply() methods now return bool and log errors properly

## Problem
UI managers (DimensionsManager, SpacingManager, etc.) could fail silently:
- No error logs
- No user notification  
- UI appears broken with no explanation
- Impossible to debug

Example of OLD CODE (v4.0.5):
```python
try:
    self._dimensions_manager.apply()
except Exception:
    pass  # SILENT FAILURE!
```

## Solution
1. Modified all `.apply()` methods to return `bool`
2. Updated callers to check return values
3. Added comprehensive error logging
4. Show warnings to users for critical failures

Example of NEW CODE (v4.0.6):
```python
try:
    success = self._dimensions_manager.apply()
    if not success:
        logger.warning("DimensionsManager failed")
        iface.messageBar().pushWarning("FilterMate", "Config error - see logs")
except Exception as e:
    logger.error(f"DimensionsManager crashed: {e}", exc_info=True)
```
