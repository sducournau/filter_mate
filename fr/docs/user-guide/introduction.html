Guide de l'utilisateur

Documentation complète de toutes les fonctionnalités et flux de travail de FilterMate.

Contenu de ce guide​

Cette section fournit une documentation détaillée pour chaque aspect de FilterMate :

Interface et Bases​

  • Aperçu de l'interface - Visite complète de l'interface et explication des onglets
  • Bases du filtrage - Filtrage par attributs avec les expressions QGIS

Opérations spatiales​

  • Filtrage géométrique - Prédicats spatiaux et relations
  • Opérations de tampon - Zones tampons et analyse de proximité

Gestion des données​

  • Exporter des entités - Formats d'export, options et traitement par lots
  • Historique des filtres - Annuler/refaire, suivi de l'historique et favoris

À qui s'adresse ce guide​

Ce guide s'adresse à :

  • 🗺️ Analystes SIG - Travaillant quotidiennement avec des données spatiales
  • 📊 Data Scientists - Filtrant et analysant des jeux de données géospatiales
  • 🔧 Développeurs - Intégrant FilterMate dans des flux de travail automatisés
  • 🎓 Étudiants et Enseignants - Apprenant QGIS et l'analyse spatiale

Prérequis​

Pour utiliser ce guide efficacement :

  • ✅ QGIS 3.x installé et en cours d'exécution
  • ✅ Plugin FilterMate installé (Guide d'installation)
  • ✅ Connaissances QGIS de base - Familiarité avec le chargement de couches et la navigation de base
  • ✅ Données d'exemple - Une couche vectorielle chargée dans votre projet

Navigation rapide​

Nouveaux utilisateurs ? Commencez par :

  1. Aperçu de l'interface - Comprendre l'interface
  2. Bases du filtrage - Apprendre le filtrage de base
  3. Exporter des entités - Sauvegarder vos résultats

Utilisateurs expérimentés ? Allez directement à :

  • Filtrage géométrique - Opérations spatiales avancées
  • Opérations de tampon - Analyse de proximité
  • Historique des filtres - Fonctionnalités pour utilisateurs avancés

Documentation connexe​

  • Premiers pas - Tutoriels pour débutants
  • Backends - Optimisation des performances
  • Avancé - Configuration et réglages
  • Dépannage - Problèmes courants et solutions
Besoin d'aide ?

Si vous rencontrez des problèmes :

  • 📖 Consultez le Guide de dépannage
  • 🐛 Signalez des bugs sur GitHub Issues
  • 💬 Posez des questions dans GitHub Discussions
