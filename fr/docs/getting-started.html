Premiers pas

Bienvenue sur FilterMate ! Ces tutoriels vous aideront à devenir productif rapidement.

Tutoriels de cette section​

Démarrage rapide​

Durée : 5 minutes

Apprenez le flux de travail essentiel :

  • Ouvrir FilterMate et sélectionner des couches
  • Créer votre premier filtre d'attributs
  • Comprendre la sélection du backend
  • Exporter les résultats filtrés

Votre premier filtre​

Durée : 10-15 minutes

Tutoriel complet étape par étape :

  • Configurer un filtre géométrique
  • Utiliser les opérations de tampon
  • Travailler avec les prédicats spatiaux
  • Réviser et exporter les résultats

Avant de commencer​

Assurez-vous d'avoir :

  • ✅ QGIS 3.x installé
  • ✅ Plugin FilterMate installé (Guide d'installation)
  • ✅ Couche vectorielle chargée dans votre projet

Conseils de performance​

Pour de meilleurs résultats avec de grands jeux de données :

  • 📦 Jeux de données moyens (<50k entités) : Spatialite/OGR fonctionnent bien
  • ⚡ Grands jeux de données (>50k entités) : Installez psycopg2 pour le support PostgreSQL
  • 🗄️ Très grands jeux de données (>1M entités) : Utilisez des couches PostGIS

Prochaines étapes​

Après avoir terminé ces tutoriels :

  1. Aperçu de l'interface - Explorez tous les composants de l'interface
  2. Bases du filtrage - Maîtrisez le filtrage d'attributs
  3. Filtrage géométrique - Opérations spatiales avancées
  4. Aperçu des backends - Comprendre l'optimisation des performances
Besoin d'aide ?

Consultez le Guide de dépannage ou visitez GitHub Issues.
